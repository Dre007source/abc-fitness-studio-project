# ABC Fitness Studio Website  

🚀 Project Overview  
This is a website designed for **ABC Fitness Studio**, a local fitness center that offers various workout programs such as **Yoga, Pilates, Zumba, and Strength Training**. The website provides an experience where users can explore classes, view available fitness gear, and sign up for membership plans.  

 📁 File Structure  
The project consists of four main HTML pages:  

| File Name | Description |
|-----------|------------|
| `index.html` | Homepage – Introduces ABC Fitness Studio and highlights class offerings. |
| `gallery.html` | Gallery Page – Displays fitness classes and training equipment with purchase options. |
| `about.html` | About Us Page – Provides business information, contact details, and an inquiry form. |
| `plans.html` | Membership Plans Page – Showcases available fitness membership plans. |

 🌟 Features  
✅ **Navigation Bar** – Easy access to all website pages.  
✅ **Interactive Call-to-Action (CTA) Buttons** – Encourages user engagement.  
✅ **Gallery with Product Descriptions** – Displays images of fitness classes and equipment.  
✅ **Membership Plans Section** – Users can compare different fitness memberships.  
✅ **Contact Form** – Allows users to reach out with inquiries.  
✅ **Mobile-Friendly & Responsive Design** – Works on all devices.

 📌 How to Use  
1. Open the homepage (`index.html`) to explore available fitness programs.  
2. Visit the **Gallery Page** to view training equipment and classes.  
3. Go to the **About Page** to learn about ABC Fitness Studio and submit an inquiry.  
4. Check out the **Plans Page** to view membership options and sign up.
